BAXZ Website Structure

As the CEO of NXH, I'm walking you through this basic structure of the BAXZ website layout. This is a lightweight HTML layout ready to be enhanced with custom CSS. Below, I explain the purpose of each section and how it contributes to the overall user experience and brand presentation.

Header

The header section is the starting point for every visitor. It introduces the brand name "BAXZ" and provides a basic navigation bar. The navigation includes links to the home page, about section, services section, and contact form. While the links are currently empty or anchored to page sections, they form the core of how users move around the site.

<div id="header"> <div class="logo"> <p>BAXZ</p> </div> <div class="nav"> <ul> <li><a href="">home</a></li> <li><a href="">about</a></li> <li><a href="#service">services</a></li> <li><a href="#contact">contact</a></li> </ul> </div> </div>

Banner

This section acts as the hero area of the page. It delivers a bold first impression with a short message and a "Get Started" button. It's a placeholder for marketing or brand messaging, and it's where users' attention is immediately drawn.

<div class="banner"> <p>Lorem ipsum dolor sit amet, consectetur, <br> adipisicing elit. Qui, cupiditate!</p> <button>get started</button> </div>

Global Styling

The first rule in our stylesheet resets padding and margin across all elements using the universal selector (*). This ensures consistency across browsers. We also set box-sizing: border-box for predictable layout behavior.

The root font size is defined as 10px, which makes rem-based sizing easier to calculate (e.g., 1.6rem = 16px). We also define two CSS variables: --main (purple) for the main brand color and --primary (yellow) for highlights and accents.

:root { --main: purple; --primary: yellow; }

Header

The header uses Flexbox to space content between the logo and the navigation. It features a bold purple background (our brand’s main color), with white navigation text that turns yellow on hover. The logo text is highlighted in yellow for contrast and brand recall.

The navigation links are capitalized for consistency and professionalism. Font sizes are increased for readability and brand presence.

#header { display: flex; justify-content: space-between; padding: 2% 5%; background-color: var(--main); } .nav a { color: white; font-size: 2rem; } .nav a:hover { color: var(--primary); }

Banner

The banner section uses a full-screen background image for visual impact. It’s styled with background-size: cover and background-position: center to keep the image looking good on all screen sizes.

The call-to-action text is styled in white, and the button inherits our purple color theme. On hover, the button stays consistent for now — but could be enhanced later with transition effects or shadow.

.banner { background-image: url(../img/pexels-tima-miroshnichenko.jpg); height: 100vh; padding: 12% 5%; background-size: cover; background-position: center; } .banner button { background-color: var(--main); color: white; padding: 10px; border-radius: 1rem; }

Services

This section showcases four service cards side-by-side using Flexbox. Each service card includes an image and a title. Cards have soft rounded corners, a purple background, and a yellow glow via box-shadow.

On hover, each card lifts slightly using transform: translateY(-15px) for a dynamic, modern effect. This adds interactivity and responsiveness to the user experience.

.serve { display: flex; justify-content: space-between; } .serve > div { background-color: var(--main); box-shadow: 0 0 30px var(--primary); transition: 0.5s; } .serve > div:hover { transform: translateY(-15px); }

Contact

The contact section is split into two columns: a form on the left and a Google Map on the right. Flexbox is used again for layout. Inputs and textarea fields are styled for size and clarity, and they receive a purple border highlight on focus.

The submit button is styled with purple as the background, white text, and a hover state that inverts colors to draw attention and indicate interactivity.

.left input, .left textarea { width: 100%; font-size: 1.6rem; } .left input:focus, .left textarea:focus { border: 2px solid var(--main); } .left button { background-color: var(--main); color: white; border-radius: 0.5rem; } .left button:hover { background-color: var(--primary); color: black; }

Footer

The footer is divided into a top and bottom part. The top features the brand, quick links, and social icons laid out in Flexbox. The bottom displays copyright info.

All text uses white for readability on a purple background, while headings and icons follow the yellow accent color. Social icons are styled with Boxicons and spaced with Flex.

#footer { background-color: var(--main); } .foot1 { display: flex; justify-content: space-between; } .foot1 i { color: var(--primary); } .foot2 { background-color: black; color: white; text-align: center; padding: 2% 0; }
